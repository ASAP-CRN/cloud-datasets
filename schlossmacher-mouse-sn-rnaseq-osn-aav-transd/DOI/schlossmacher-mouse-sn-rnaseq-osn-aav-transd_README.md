This Zenodo deposit contains a publicly available description of the Dataset:

# "snRNAseq of olfactory epithelium with AAV serotypes for transduction of olfactory sensory neurons in Mus musculus".

## Dataset Description:
 
Single-nucleus RNA sequencing data of 4 mice, each inoculated with 4 AAV serotypes. The mice were all male and 8-10 weeks of age at when olfactory epithelium tissue was harvested. Samples were paired-end sequences using v3.1 – Dual Index using Illumina Novaseq 6000 sequencer.
